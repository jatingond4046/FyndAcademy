Bold VS Strong tag

Both tags are formatting tags and used to bold the text but bold tag is usually used to bold the topic or headings or subheadings and strong tag is used to draw the attention towards the text that it is important

Example:

Introduction : My name is Jatin Kumar and I live in katni

Here Introduction is jsut heading so it is covered with bold tag and Jatin kumar is the name so it is important so it is covered with strong tag

Strike VS Delete tag

The delete tag defines text that has been deleted from a document. Browsers will usually strike a line through deleted text.

Use the strike element to represent things that are no longer relevant or no longer accurate

Example :

My favorite color is blue red!

There will be a few tickets available at the box office tonight.

SOLD OUT!
